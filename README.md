# Dataset for: Spatiotemporal dynamics and driving mechanisms of ecosystem service value in an arid oasis

## Description
This repository provides the processed datasets supporting the analysis of ecosystem service value (ESV) in an arid oasis. The data included here are analysis-ready and derived from multiple publicly available sources.

## Data Sources
The original datasets used in this study are publicly accessible from the following platforms:

- Remote sensing data (land use and NDVI): NASA LAADS DAAC  
  https://ladsweb.modaps.eosdis.nasa.gov

- Digital Elevation Model (DEM): Geospatial Data Cloud  
  https://www.gscloud.cn

- Meteorological data (precipitation, temperature, evapotranspiration) and partial socioeconomic data: National Earth System Science Data Center  
  https://www.geodata.cn

- Geographic data (water bodies, residential areas, roads, administrative boundaries): National Catalogue Service for Geographic Information  
  https://www.webmap.cn

- Statistical data: China Statistical Yearbook (2005–2025) and National Compilation of Agricultural Product Cost and Income Data (2005–2025), published by the National Development and Reform Commission  
  https://www.ndrc.gov.cn

## Repository Contents
- `data/`: processed datasets used in analysis  
- `results/`: key outputs (e.g., ESV results)  
- `code/`: scripts for data processing and analysis (if applicable)

## Data Availability
Due to data licensing and size constraints, raw datasets are not redistributed in this repository. Processed data supporting the findings of this study are included. Additional data and materials are available from the corresponding author upon reasonable request.
